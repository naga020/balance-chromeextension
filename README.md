# Balance: Mindful Browsing

A Chrome extension that helps users reduce distractions, track screen time, and build healthier browsing habits.

## Features
- Site blocking with motivational nudges
- Pomodoro focus timer
- Daily usage reports
- Smart blocking scheduler
- Distraction history log

## Installation
1. Clone this repo
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Load unpacked and select the project folder
